# 📈 Product Price Prediction

Product Price Prediction is a machine learning web application that predicts the estimated price of a product based on its features. The project uses data preprocessing, feature engineering, and regression models to generate accurate price predictions through an intuitive user interface.

## ✨ Features

- 🤖 Machine Learning-based price prediction
- 📊 Data preprocessing and feature engineering
- 📈 Regression model for price estimation
- 📝 User-friendly prediction interface
- ⚡ Fast and accurate predictions
- 📱 Responsive web application

## 🚀 Installation

### Clone the repository

```bash
git clone https://github.com/sanjaychavan05/Product-Price-Prediction.git
```

### Navigate to the project

```bash
cd Product-Price-Prediction
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Run the backend

```bash
python app.py
```

### Run the frontend

```bash
npm install
npm run dev
```

## 📖 How It Works

1. Upload or enter product details.
2. The application preprocesses the input data.
3. The trained machine learning model predicts the product price.
4. The estimated price is displayed to the user.
